How to setup Google Apps Script (100% Free):

  1. Go to Google Apps Script and create a new project.
  2. Delete any default code and paste the script code below.
  3. Click Deploy > New Deployment. Choose Web app.
  4. Set Execute as to Me and Who has access to Anyone.
  5. Click Deploy, authorize permissions, and copy the Web App URL into the settings form.

function doPost(e) {
  try {
    var params = JSON.parse(e.postData.contents);
    var email = params.email;
    var subject = params.subject;
    var message = params.message;
    
    MailApp.sendEmail({
      to: email,
      subject: subject,
      body: message
    });
    
    return ContentService.createTextOutput(JSON.stringify({ success: true }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({ success: false, error: error.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

How to setup EmailJS:

  1. Sign up at emailjs.com.
  2. Add your email service (e.g., Gmail) to get a Service ID.
  3. Create an email template named Valley Confirmation.
  4. In the template, set recipient email to {{to_email}} and body to {{message}}.
  5. Get your Public Key from account settings, paste all IDs here, and save!

Web3Forms Auto-Responder Note:

By default, Web3Forms forwards customer orders directly to the store owner's inbox.

To send automatic confirmation emails back to the customer's email ID, Web3Forms requires a Pro/Paid Subscription to enable their auto-responder feature.

For a 100% free solution, choose the Google Apps Script method.
